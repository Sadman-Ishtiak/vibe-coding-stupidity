# Internova Feature Implementation Log - Jan 10, 2026

## 1. Navigation Enhancements
- Added a "Home" button to the Desktop and Mobile navigation bar.
- Updated the theme toggle and mobile menu buttons to be circular (`rounded-full`) to match the UI style.

## 2. Page & Routing Updates
- Connected "Find an Internship" and "Post a Job" buttons on the `/about` page to `/jobs` and `/company` respectively.
- Commented out the non-existent `/blog` link in the footer.

## 3. Search & Discovery
- Connected the homepage search bar (Title and Location) to the `/jobs` page.
- Implemented backend support for location-based filtering.
- Made homepage Job Categories dynamic with live counters for active circulars, sorted by popularity.
- Clicking a category now redirects to a filtered job list.

## 4. Job & Company Data Models
- Refined the relationship between "Industry" (Company-level) and "Job Category" (Job-level).
- Jobs now automatically inherit the Industry from the Company profile.
- Added a detailed "Description" field to job circulars with support for line breaks (`whitespace-pre-line`).
- Optimized the "Post Job" form layout into a clean 2x2 grid for basic info.

## 5. Application Management
- Implemented a status management system for applications: `Pending`, `Accepted`, `Rejected`.
- Created a "My Applications" section for users to track their status and read employer messages.
- Added a mandatory messaging requirement for employers when accepting or rejecting candidates.

## 6. Trust, Safety & Risk Analysis
- Implemented an automated Risk Analysis dashboard for admins.
- **Ghost Circular Detection:** Flags active jobs with applicants but no employer activity for 30+ days.
- **Spam Bot Detection:** Flags users with high application velocity.
- **Dynamic Limits:** User application limits are now boosted based on their certifications (Professional, Academic, etc.).
- **Ghost Rating System:** Companies get a `ghostStrikeCount` when their ghost circulars are verified and deleted by admins.
- **Prioritization:** The job feed now automatically deprioritizes circulars from companies with high ghost strike counts.

## 7. Skill Search Enhancements
- Fixed the skill filter on the `/people` page by adding state management and interactive toggle buttons.
- Upgraded the skill filter to a **typable dropdown (Combobox)** for better UX.
- Implemented a backend API endpoint (`/api/stats/skills`) to fetch and suggest the top 50 most popular skills dynamically.

## 8. Authentication & Google OAuth
- Validated readiness for Google OAuth login and signup.
- Added a 'Sign up with Google' button to the registration page.
- Configured `.env.local` with the provided Google Client ID and Secret.
- Resolved 'Server error' by generating a secure `AUTH_SECRET` and configuring `NEXTAUTH_URL` for deployment.
- Instructed on configuring the 'Authorized redirect URIs' in Google Cloud Console to fix `redirect_uri_mismatch`.

## 9. UI/UX Refinements (Dark Mode & Visibility)
- Improved text visibility in dark mode across the entire application (Home, Jobs, Companies, Profile, Admin).
- Lightened colors for 'indigo', 'blue', 'green', and 'red' text elements to ensure readability against dark backgrounds.
- Updated Navbar dropdown text colors (Logout & Admin Panel) for better contrast.

## 10. Print Optimization
- Enhanced the CV print view for User Profiles.
- Replaced the 'View Credential' hyperlink with the full, visible URL specifically in print mode, ensuring certifications are verifiable on physical copies.
